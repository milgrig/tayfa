# Changelog

История значимых изменений продукта Tayfa.

---

## [Unreleased]

### Добавлено
- Система обсуждений `.tayfa/common/discussions/` — централизованное место для коммуникации по задачам
- Обязательные форматы handoff между этапами (постановщик → разработчик → тестировщик)
- Папка skills/ для каждого агента — методологии и шаблоны
- Продуктовая документация `docs/product/`
- Правила ведения документации `product-docs.md`

### Изменено
- Удалены папки `income/` и `done/` — заменены на `discussions/`
- Обновлены промпты всех агентов с новыми форматами handoff
- Обновлены правила `teamwork.md`

### Удалено
- Система входящих/исходящих заданий через `income/` и `done/`

---

## [0.1.0] - Начальная версия

### Добавлено
- Система задач и спринтов с тремя ролями
- Оркестратор (Web UI) на FastAPI
- Интеграция с Claude Code CLI
- Автоматические git-операции
- Управление агентами через employees.json
- Базовые агенты: boss, hr, analyst, developer, tester, process_architect
- Правила работы команды (teamwork.md, employees.md, git-workflow.md)
- Skills для boss и process_architect
