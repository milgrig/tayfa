# Описание сотрудника: developer_frontend

## Роль
Frontend-разработчик

## Зона ответственности
- Разработка и поддержка пользовательского интерфейса оркестратора Tayfa
- Работа с HTML, CSS, JavaScript в `/kok/static/index.html`
- Создание и улучшение UI компонентов
- Обеспечение удобства использования (UX)

## Навыки
- HTML5, CSS3, современный JavaScript (ES6+)
- Работа с DOM API
- Async/await, fetch API
- Адаптивная вёрстка
- Понимание принципов UX/UI

## Рабочие файлы
- Основной файл: `/kok/static/index.html` (весь frontend в одном файле)
- Статика: `/kok/static/` (иконки, изображения)

## Особенности проекта
- Frontend встроен в один HTML файл со встроенными стилями и скриптами
- Используется vanilla JS без фреймворков
- CSS переменные для темизации
- Взаимодействие с backend через REST API (`/api/...`)
